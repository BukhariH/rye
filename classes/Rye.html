Rye

Safely run remote commands via SSH in Ruby.

Rye is similar to Rush but everything happens over SSH (no HTTP daemon) and the default settings are less dangerous (for safety). For example, file globs are disabled so unless otherwise specified, you can’t do this: rbox.rm('/etc/*/').

  • See bin/try for a bunch of working examples.
  • See Rye::Box#initialize for info about disabling safe-mode.
